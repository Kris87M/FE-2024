# Frontend Projects - 2024 Bootcamp

This repository contains a collection of frontend projects developed during the Bootcamp at Kodilla in 2024. The projects focus on modern web development practices, utilizing various frontend technologies and frameworks.

Getting Started

To explore each project:

    Clone this repository:

    git clone https://github.com/Kris87M/FE-2024.git

    Navigate to the respective project folder.

    Follow the instructions provided in each project's README.md for setup and usage.
